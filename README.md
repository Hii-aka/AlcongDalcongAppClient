# 알콩달콩 (AlkongDalkong) - 커플 다이어리 앱 📱💕

## 소개
알콩달콩은 연인들을 위한 스마트한 다이어리 앱입니다. 소중한 추억을 기록하고, 데이트 일정을 관리하며, AI 기반의 데이트 장소 추천 기능을 제공합니다.

## 주요 기능 ✨

### 1. 커플 다이어리
- 일상 기록 및 공유
- 사진 및 동영상 첨부
- 감정 이모티콘으로 그날의 기분 표현
- 실시간 동기화로 파트너와 즉시 공유

### 2. 데이트 캘린더 📅
- 데이트 일정 등록 및 관리
- 장소 정보 저장
- 알림 설정
- 기념일 자동 계산 및 리마인더

### 3. AI 데이트 플래너 🤖
- 취향 기반 데이트 코스 추천

## 기술 스택 🛠

- Frontend: React Native, Expo
- State Management: Zustand
- Data Fetching: React Query, Axios
- UI/UX: Tailwind CSS
- AI/ML: OpenAI (llm)
- Database: Firebase
- Authentication: 
  - Kakao Login
  - Apple Login
